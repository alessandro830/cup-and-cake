#Cup&Cake

## Descrição
O projeto é uma vitrine de uma loja de doces, onde é possível realizar um CRUD no produto e depoimentos dos clientes, além de uma página de cadastro e login.

##Tecnologias
- NextJs
- CSS Modules
- TypeScript
``

## Instrução de Instalação

### Pré-requisitos

- Node.js
- npm
- Git

```bash
git clone https://github.com/alessandro830/cup-and-cake.git
cd cup-and-cake
npm install
```

###Como executar o projeto

```bash
npm run dev
```
Após executar o comando, basta abrir o localhost. Geralmente http://localhost:3000/.

O projeto também pode ser acessado pelo Vercel: [httpscup-and-cake.vercel.app](https://cup-and-cake.vercel.app/)
